# 获取未导出数据

> **所需套餐：** 标准版、专业版或企业版

## 端点

```
GET https://openapi.octoparse.com/data/get-non-exported-data
```

## 参数

| 参数 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `taskGroupId` | string | 是 | 任务组 ID |

## 响应

返回尚未标记为已导出的记录。用于获取新提取的数据。

另请参阅：[按偏移量获取数据](/en/api-reference/data/get-data-by-offset)、[标记数据为已导出](/en/api-reference/data/mark-data-as-exported)
