# 标记数据为已导出

> **所需套餐：** 标准版、专业版或企业版

## 端点

```
POST https://openapi.octoparse.com/data/mark-exported
```

## 请求体

| 参数 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `taskGroupId` | string | 是 | 任务组 ID |
| `recordIds` | array | 是 | 要标记为已导出的记录 ID |

另请参阅：[获取未导出数据](/en/api-reference/data/get-non-exported-data)
