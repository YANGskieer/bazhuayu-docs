# 认证

所有 Octoparse API 请求均需使用 Access Token 进行认证。

## 获取 Token

### 第一步：获取 Access Token

```bash
curl -X POST https://openapi.octoparse.com/token \
  -H "Content-Type: application/json" \
  -d "{\"username\": \"YOUR_EMAIL\", \"password\": \"YOUR_PASSWORD\"}"
```

### 第二步：在 API 请求中携带 Token

```
Authorization: Bearer YOUR_ACCESS_TOKEN
```

## Token 刷新

Access Token 会过期。使用 Refresh Token 获取新的：

```bash
curl -X POST https://openapi.octoparse.com/token/refresh \
  -H "Content-Type: application/json" \
  -d "{\"refresh_token\": \"YOUR_REFRESH_TOKEN\"}"
```

## 重要提示

- 切勿将凭据硬编码在代码中。请使用环境变量或密钥管理器。
- Access Token 与你的账户绑定，请勿分享。
- 如怀疑 Token 已被泄露，请从 Octoparse 账户设置中重置。

另请参阅：[获取新 Token](/en/api-reference/access-token/obtain-a-new-token)、[刷新 Token](/en/api-reference/access-token/refresh-token)
