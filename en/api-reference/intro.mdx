# 八爪鱼 开放接口文档

八爪鱼采集器提供开放 API 接口，支持以编程方式管理采集任务、运行云端采集、获取采集数据，方便开发者进行系统集成与自动化。

## 文档版本

当前版本：**1.0.0**

## URI 规范

基础 URL：

```
https://openapi.bazhuayu.com
```

## 接口说明

- 所有接口均使用 HTTPS 协议
- 请求体与返回体均为 JSON 格式
- **Content-Type**: `application/json`
- 每次请求必须携带认证令牌（详见 [接口调用凭证](/en/api-reference/access-token/obtain-a-new-token)）

## 权限描述

| 服务 | 接口 | 权限 |
|------|------|------|
| 接口调用凭证 | 获取全新Token、刷新Token | 全部 |
| 任务组 | 获取用户的任务组信息 | 旗舰版、旗舰+、企业版、团队版 |
| 任务组 | 批量删除任务组 | 旗舰版、旗舰+、企业版、团队版 |
| 任务 | 搜索任务、复制任务 | 旗舰版、旗舰+、企业版、团队版 |
| 任务 | 移动任务、批量删除任务、批量复制任务到多个任务组 | 旗舰+、企业版、团队版 |
| 任务 | 批量获取任务步骤信息 | 旗舰+、企业版、团队版 |
| 任务 | 更新任务参数、更新步骤配置、更新循环步骤的内容、更新采集Urls | 旗舰+、企业版、团队版 |
| 云采集 | 全部 | 旗舰版、旗舰+、企业版、团队版 |
| 数据 | 全部 | 旗舰版、旗舰+、企业版、团队版 |

## 公共错误码

| 状态码 | 错误码 | 错误信息 |
|--------|--------|----------|
| 400 | `InvalidParameter` | 字段错误 |
| 400 | `InvalidTaskId` | taskId 不能为空 |
| 400 | `InvalidSize` | size 应该在 1 和 1000 之间 |
| 400 | `InvalidActions` | actions 必须大于 0 |
| 400 | `InvalidTasks` | tasks 不能超过 100 |
| 400 | `LimitTraffic` | 超出流量限制，将size参数设置小一些再请求 |
| 400 | `RequestModelNull` | 请求参数不能为空 |
| 401 | `Unauthorized` | access_token 无效 |
| 403 | `PermissionDenied` | 没有接口访问权限 |
| 429 | `TooManyRequests` | 调用频率过快 |
| 500 | `ServerError` | 服务内部错误 |
| 503 | `ServiceUnavailable` | 服务器繁忙 |

## 遇到问题

每次响应均包含 `requestId`。如请求失败，请将其提供给 [八爪鱼支持团队](mailto:support@octoparse.com)。
