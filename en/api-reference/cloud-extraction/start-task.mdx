# 启动任务

> **所需套餐：** 专业版或企业版

## 端点

```
POST https://openapi.octoparse.com/cloud-extraction/start-task
```

## 请求体

| 参数 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `taskGroupId` | string | 是 | 要启动的任务组 ID |
| `urls` | array | 否 | 要采集的特定 URL（覆盖任务 URL） |

## 响应

| 字段 | 类型 | 说明 |
|---|---|---|
| `runId` | string | 用于追踪此次提取的运行 ID |
| `status` | string | 运行的初始状态 |

另请参阅：[停止任务](/en/api-reference/cloud-extraction/stop-task)、[获取任务状态](/en/api-reference/cloud-extraction/get-task-status)
