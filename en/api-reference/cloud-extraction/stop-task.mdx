# 停止任务

> **所需套餐：** 专业版或企业版

## 端点

```
POST https://openapi.octoparse.com/cloud-extraction/stop-task
```

## 请求体

| 参数 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `taskGroupId` | string | 是 | 要停止的任务组 ID |

另请参阅：[启动任务](/en/api-reference/cloud-extraction/start-task)
