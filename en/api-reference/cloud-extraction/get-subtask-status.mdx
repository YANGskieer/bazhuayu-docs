# 获取子任务状态

> **所需套餐：** 专业版或企业版

## 端点

```
GET https://openapi.octoparse.com/cloud-extraction/subtask-status
```

## 参数

| 参数 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `taskGroupId` | string | 是 | 任务组 ID |
| `subtaskId` | string | 是 | 子任务 ID |

## 响应

| 字段 | 类型 | 说明 |
|---|---|---|
| `subtaskId` | string | 子任务 ID |
| `status` | string | pending / running / completed / failed |
| `progress` | number | 进度百分比（0-100） |
| `recordsExtracted` | number | 目前已提取的记录数 |
| `errorMessage` | string | 失败时的错误信息 |

另请参阅：[启动任务](/en/api-reference/cloud-extraction/start-task)、[获取任务状态](/en/api-reference/cloud-extraction/get-task-status)
