# 刷新 Token

> **所需套餐：** 所有用户

## 端点

```
POST https://openapi.octoparse.com/token/refresh
```

## 描述

使用 Refresh Token 获取新的 Access Token，无需重新输入凭据。

## 请求体

| 参数 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `refresh_token` | string | 是 | 认证时获得的 Refresh Token |

## 响应

| 字段 | 类型 | 说明 |
|---|---|---|
| `access_token` | string | 新的 Access Token |
| `refresh_token` | string | 新的 Refresh Token |
| `expires_in` | number | 新 Access Token 过期前的秒数 |

## 注意事项

- Refresh Token 也会过期。请在当前 Refresh Token 过期前进行刷新。
- 如果 Refresh Token 已过期，必须通过 [获取新 Token](/en/api-reference/access-token/obtain-a-new-token) 重新获取。

另请参阅：[获取新 Token](/en/api-reference/access-token/obtain-a-new-token)
