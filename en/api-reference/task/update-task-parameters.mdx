# 更新任务参数

> **所需套餐：** 旗舰+、企业版、团队版

## 端点

```
https://openapi.bazhuayu.com/task/updateTaskParameters
```

## 描述

通过该接口，用户可以一次性更新多行输入框中的可循环参数（如多行文本、多行URLs），以及单个文本、数字等非循环参数。

taskId
required

任务Id

required
	Object[]

required
	String	

步骤类型，目前支持：NavigateAction（打开网页类型）、EnterTextAction（输入文本类型）、LoopAction（循环列表类型）


actionId
required
	String	

步骤的唯一标识，您可以在客户端点击相应的步骤的"Api 调用名称"获取，也可以通过【批量获取任务步骤信息】接口获取。

required
	Object[]

required

value
required
	String	

属性值


loopItems
required
	Object[]	

需要更新的循环项参数


actionId
required
	String	

步骤的唯一标识，您可以在客户端点击相应的步骤的"Api 调用名称"获取，也可以通过【批量获取任务步骤信息】接口获取。


loopType
required
	String	

循环类型，目前支持：URLList（网址列表）、TextList（文本列表）


loopItems
required
	String[]	

循环项内容


isAppend
required
	Boolean	

是否是追加流程值，只针对List型流程值，如果是追加，则不清除旧值，默认清除旧值

| `请求Id` |  | 否 |  |
| `error` |  | 否 |  |
| `required` | 错误信息，参考错误码 | 是 |  |
| `code` |  | 否 |  |
| `required` |  | 是 |  |
