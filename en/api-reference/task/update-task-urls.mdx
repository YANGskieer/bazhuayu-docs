# 更新任务 URL

> **所需套餐：** 专业版或企业版

## 端点

```
POST https://openapi.octoparse.com/task/update-urls
```

## 请求体

| 参数 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `taskGroupId` | string | 是 | 任务组 ID |
| `urls` | array | 是 | 新的目标 URL 列表 |

另请参阅：[搜索任务](/en/api-reference/task/search-task)、[复制任务](/en/api-reference/task/duplicate-task)
