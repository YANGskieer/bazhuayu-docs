# 复制任务

> **所需套餐：** 标准版、专业版或企业版

## 端点

```
POST https://openapi.octoparse.com/task/duplicate
```

## 请求体

| 参数 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `taskGroupId` | string | 是 | 要复制的任务组 ID |
| `newName` | string | 否 | 新任务的名称 |

## 响应

返回新创建任务的 ID 和名称。

另请参阅：[搜索任务](/en/api-reference/task/search-task)、[更新任务 URL](/en/api-reference/task/update-task-urls)
