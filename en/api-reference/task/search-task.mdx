# 搜索任务

> **所需套餐：** 标准版、专业版或企业版

## 端点

```
GET https://openapi.octoparse.com/task/search
```

## 参数

| 参数 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `taskGroupId` | string | 否 | 按任务组 ID 筛选 |
| `keyword` | string | 否 | 按任务名称搜索 |

## 响应

返回匹配的任务列表，包含任务 ID、名称和状态。

另请参阅：[复制任务](/en/api-reference/task/duplicate-task)、[更新任务 URL](/en/api-reference/task/update-task-urls)
