# 获取任务组信息

> **所需套餐：** 标准版、专业版或企业版

## 端点

```
GET https://openapi.octoparse.com/task-group/get-information
```

## 参数

| 参数 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `taskGroupId` | string | 是 | 任务组 ID |

## 响应

返回任务组的详细信息，包括名称、创建日期、状态和配置。

另请参阅：[搜索任务](/en/api-reference/task/search-task)
