# MCP 对接 Coze（扣子）流程

> 在 Coze（扣子）中使用八爪鱼 MCP：推荐优先使用官方插件，也可自定义 MCP 服务。

## 前置准备

- 扣子 / Coze 账户
- 八爪鱼账户（已完成登录）
- MCP 地址：`https://mcp.bazhuayu.com`
- 若选择 API Key 模式，请先 [获取 API Key](/en/mcp/quick-start/api-key)

## 方式一：官方插件（推荐）

1. 打开 [扣子编程](https://code.coze.cn/home)，进入 **旧版** 工作区。
2. 进入 **插件商店**，搜索 **「八爪鱼采集器」**。
   - 也可直接打开官方插件页：[八爪鱼采集器插件](https://www.coze.cn/store/plugin/7638892610710421504?from=store_search_suggestion)
3. 安装插件后，按引导选择连接方式：
   - **OAuth 授权**：浏览器登录八爪鱼并允许访问（无需手动复制 Key）
   - **API Key 授权**：粘贴你在 [账户中心](/en/mcp/quick-start/api-key) 创建的 Key

安装完成后，在 Bot / 工作流中启用该插件即可在对话中采集数据。

## 方式二：自定义添加 MCP

适用于需要自行管理 MCP 端点或使用固定 API Key 的场景。

### 步骤 1：进入资源库

1. 登录 [扣子编程](https://code.coze.cn/home)，切换至 **旧版**。
2. 打开 **资源库**，点击右侧 **添加** → 选择 **插件**。

### 步骤 2：填写插件信息

在配置窗口中填写：

| 字段 | 建议值 |
| --- | --- |
| 插件名称 | 八爪鱼采集器助手（可自定义） |
| 一句话介绍 | 八爪鱼官方 MCP 数据采集 |
| 类型 | **MCP**（固定，不可更改） |
| 插件 URL | `https://mcp.bazhuayu.com` |

图标、头像可按团队规范自定义上传。

### 步骤 3：配置认证

保存后进入插件详情，配置 MCP 连接：

- **OAuth**：按扣子提示完成八爪鱼登录授权  
- **API Key**：在 Headers 中设置 `x-api-key` 为你的 Key  

### 步骤 4：在应用中使用

1. 创建或编辑 Bot / 工作流。  
2. 在 **插件 / 工具** 中勾选已添加的八爪鱼 MCP。  
3. 发布后在对话中测试，例如：「搜索一个采集商品评论的八爪鱼模板并运行」。

## 常见问题

| 现象 | 处理建议 |
| --- | --- |
| 插件列表找不到八爪鱼 | 确认在「插件商店」搜索全称；或使用方式二自定义 MCP |
| 授权后无工具 | 断开 MCP 重新连接；确认 URL 无多余斜杠 |
| 仅模板搜索成功、运行失败 | 模板可能仅支持本地采集，请换云采集模板或在桌面客户端运行 |
