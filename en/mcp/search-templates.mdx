# search_templates

按关键词或类别搜索 Octoparse 预置采集模板。

## 参数

| 参数 | 类型 | 说明 |
|---|---|---|
| `query` | string | 搜索关键词或短语 |
| `limit` | number | 最大结果数量（默认：10） |

## 响应

返回匹配的模板列表，包含名称、描述、目标网站、支持的页面类型及最后更新日期。

另请参阅：[MCP 概览](/en/mcp/intro)、[工作流程示例](/en/mcp/workflow)
