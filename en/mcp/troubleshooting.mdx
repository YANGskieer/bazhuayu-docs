# MCP 故障排查

> 针对常见连接、授权及任务相关问题的修复方案。

## 连接问题

### MCP 服务端无法访问

**症状：** 调用 MCP 工具时出现 `ECONNREFUSED` 或 `ETIMEDOUT`。

**修复方案：**
- 确认服务器地址正确：`https://mcp.octoparse.com`
- 检查你的网络连接和防火墙规则
- 确认 MCP 服务端未发生故障

### 响应时间过慢

**症状：** 工具调用超过 30 秒才返回。

**修复方案：**
- 大批量任务提取需要时间；适当增加超时设置
- 检查你到 MCP 服务端的网络延迟
- 考虑将大型提取任务拆分为更小的批次

## 授权问题

### API Key 无效

**症状：** `401 Unauthorized` 或 `403 Forbidden` 响应。

**修复方案：**
- 确认你的 API Key 正确且处于激活状态
- 在 [Octoparse 账户中心](https://www.octoparse.com/console/account-center/api-keys) 重新生成 Key
- 确认 Key 未过期

### OAuth 未连接

**症状：** OAuth 认证提示失败或循环。

**修复方案：**
- 确认你已登录正确的 Octoparse 账户
- 清除浏览器 Cookie 后重新授权
- 在无头环境下尝试使用 API Key 认证替代

## 任务执行问题

### 任务未找到

**症状：** 调用 `start_or_stop_task` 时返回 `Task not found`。

**修复方案：**
- 确认任务 ID 正确
- 确认该任务存在于你的 Octoparse 账户中
- 检查任务是否设置为云端提取（而非仅本地）

### 无数据返回

**症状：** `execute_task` 完成但返回空数据。

**修复方案：**
- 任务可能仍在运行中；稍等后重试
- 在 Octoparse 控制面板中确认任务已成功完成
- 确认目标网站可从 Octoparse 云端服务器访问

### 导出格式不支持

**症状：** 调用 `export_data` 指定特定格式时出错。

**修复方案：**
- 支持的格式为 `json` 和 `csv`
- 确认格式参数为小写
- 在调用导出前确认任务有数据可供导出

## 获取帮助

如果问题持续存在，请联系 [Octoparse 支持团队](mailto:support@octoparse.com)，并提供：
- 失败的 MCP 工具名称
- 完整的错误信息和请求 ID
- 你的 Octoparse 账户邮箱
