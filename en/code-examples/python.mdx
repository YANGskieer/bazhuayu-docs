# Python 代码示例

以下代码示例展示了如何使用 Python 调用八爪鱼开放 API。

## 准备工作

```bash
pip install requests
```

## 获取 Access Token

```python
import requests

BASE_URL = "https://openapi.bazhuayu.com"

# 获取Access Token
response = requests.post(
    f"{BASE_URL}/token",
    json={
        "username": "your_username",
        "password": "your_password",
        "grant_type": "password"
    },
    headers={"Content-Type": "application/json"}
)

result = response.json()
access_token = result["data"][0]["access_token"]
print(f"Access Token: {access_token}")
```

## 获取任务组信息

```python
response = requests.get(
    f"{BASE_URL}/taskGroups",
    headers={
        "Authorization": access_token,
        "Content-Type": "application/json"
    }
)

groups = response.json()["data"]
for group in groups:
    print(f"任务组: {group['taskGroupName']} (ID: {group['taskGroupId']})")
```

## 搜索任务

```python
response = requests.post(
    f"{BASE_URL}/tasks/search",
    json={
        "pageSize": 10,
        "pageIndex": 0
    },
    headers={
        "Authorization": access_token,
        "Content-Type": "application/json"
    }
)

tasks = response.json()["data"]
for task in tasks:
    print(f"任务: {task['taskName']} (ID: {task['taskId']})")
```

## 启动云采集任务

```python
response = requests.post(
    f"{BASE_URL}/tasks/start",
    json={"taskId": "your_task_id"},
    headers={
        "Authorization": access_token,
        "Content-Type": "application/json"
    }
)

print(f"任务启动状态: {response.json()}")
```

## 获取采集数据

```python
response = requests.post(
    f"{BASE_URL}/data/getDataByOffset",
    json={
        "taskId": "your_task_id",
        "offset": 0,
        "size": 100
    },
    headers={
        "Authorization": access_token,
        "Content-Type": "application/json"
    }
)

data = response.json()
print(f"获取到 {len(data['data'])} 条数据")
```
