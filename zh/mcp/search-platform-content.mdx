按关键词搜索八爪鱼数据中心的存量内容，可限制时间范围、包含或排除指定域名。该工具是只读操作，不会创建新的采集任务。

## 什么是数据中心

八爪鱼通过对接各大平台的数据接口与公开内容源，提前持续采集、清洗并索引公开内容。调用 `search_platform_content` 时，系统直接查询已经积累的数据，而不是收到请求后再临时抓取目标网站，因此更适合 Agent 做快速、跨平台和可重复的资料检索。

数据中心的优势体现在三个方面：

- **数量级**：面向持续增长的大规模存量内容库；覆盖数百个国内外网站，而不是一次调用只采一个页面
- **广度**：覆盖社交社区、新闻资讯、财经证券、汽车、消费维权、招投标和海外媒体等内容源，以及 20+ 语种
- **宽度**：同一入口可检索网页、图片和视频，并按域名、时间范围、关键词组合筛选

具体记录总量会随持续采集变化。平台示例和当前可查询域名见 [`list_platforms`](/zh/mcp/list-platforms)，实时范围以工具返回为准。

## 适用场景

- **品牌舆情监控**：按品牌词和时间窗口跨平台聚合公开讨论
- **竞品与行业追踪**：同时检索社交、新闻、财经和垂直站点的动态
- **事件多源研究**：对照不同地区、语种和平台的报道口径
- **商机线索发现**：定向检索招投标、政府采购和行业资讯来源

## 参数

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `query` | string | 是 | 搜索词，长度 `1-500` |
| `freshness` | string | 否 | 时间范围，格式见下方 |
| `summary` | boolean | 否 | 是否返回网页摘要，默认 `true` |
| `include` | string | 否 | 仅搜索指定域名；多个域名用 `\|` 或逗号分隔 |
| `exclude` | string | 否 | 排除指定域名；多个域名用 `\|` 或逗号分隔 |
| `count` | integer | 否 | 最大结果数，范围 `1-50`，默认 `10` |
| `roleCode` | string | 否 | 搜索角色代码，默认 `test1`；没有明确要求时建议省略 |
| `needCount` | boolean | 否 | 是否请求精确结果总数，默认 `false` |

`freshness` 支持：`noLimit`、`oneDay`、`oneWeek`、`oneMonth`、`oneYear`、单日 `YYYY-MM-DD`，或闭区间 `YYYY-MM-DD..YYYY-MM-DD`。

```json
{
  "query": "人工智能行业动态",
  "freshness": "oneWeek",
  "include": "example.com|news.example.org",
  "count": 10,
  "summary": true
}
```

## 响应

返回 `success`、`code`、`message`、`data` 和 `timestamp`。`data` 可包含网页、图片或视频结果，具体字段随上游内容类型扩展，客户端应先检查实际返回结构，再展示标题、来源、链接、摘要和媒体信息。

<Note>
  搜索范围会随数据中心索引变化。需要确认支持的平台或域名时，先调用 [`list_platforms`](/zh/mcp/list-platforms)。
</Note>
