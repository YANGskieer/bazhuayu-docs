> 通过 npm 安装八爪鱼 CLI，并确认 `octopus` 命令、浏览器依赖与本地运行环境可用。

八爪鱼 CLI 以 npm 包形式发布。v0.1.28 新增任务组、模板任务、云定时、数据计数与预览等远端管理能力；v0.1.27 的 Agent 视觉规划、API 列表候选和字段名规范化能力继续适用。

## 环境要求

安装前请确认：

- Node.js **20 或更高**（推荐 22 LTS）
- npm **8 或更高**
- 可用终端：PowerShell、CMD、macOS 终端或 Linux Shell
- 网页 `detect` 与本地 `run` 需要可用 Chrome 环境

检查版本：

```bash
node -v
npm -v
```

## 平台支持

| 平台 | 本地采集 | 云端采集 | 备注 |
| --- | :---: | :---: | --- |
| Windows x64 | 支持 | 支持 | Windows 10 / 11 |
| macOS x64 | 支持 | 支持 | 需 Node.js 20+ |
| macOS arm64 | 支持 | 支持 | 需 Node.js 20+ |
| Linux x64 | 支持 | 支持 | v0.1.22 新增 |
| Linux arm64 | 暂不支持 | 支持 | Chrome for Testing 暂无 Linux arm64 浏览器包 |

## 安装 Node.js

### Windows

1. 访问 [Node.js 官网](https://nodejs.org/) 下载 LTS 安装包。
2. 使用默认选项安装，并确认勾选加入 `PATH`。
3. 打开 PowerShell 或 CMD，执行：

```bash
node --version
npm --version
```

### macOS

可从 [Node.js 官网](https://nodejs.org/) 下载 macOS 安装包，或使用 Homebrew：

```bash
brew install node
node --version
npm --version
```

### Linux / 服务器

推荐使用 [nvm](https://github.com/nvm-sh/nvm) 安装 Node.js 20+：

```bash
node -v
npm -v
```

Linux x64 从 CLI v0.1.22 起支持本地采集；Linux arm64 目前仅建议使用云采集相关能力。

## 安装八爪鱼 CLI

安装或更新到 v0.1.28：

```bash
npm install -g bazhuayu-cli@0.1.28
```

安装完成后，命令名为：

```bash
octopus
```

查看版本与帮助：

```bash
octopus --version
octopus --help
```

v0.1.28 应能看到 `detect`、`task-group`、`template`、`schedule` 和 `data preview` 等命令：

```bash
octopus detect --help
octopus task-group --help
octopus data preview --help
```

## 运行环境诊断

```bash
octopus doctor
octopus browser doctor
```

`doctor` 会检查 Node.js、内置引擎、Chrome 可用性及浏览器模式。成功时通常返回 `"ok": true`。

## 更新与卸载

更新到指定版本：

```bash
npm install -g bazhuayu-cli@0.1.28
octopus --version
```

卸载：

```bash
npm uninstall -g bazhuayu-cli
```

## 常见问题

| 现象 | 处理建议 |
| --- | --- |
| `octopus: command not found` | 将 Node.js 目录和 npm 全局 bin 目录加入系统 `PATH` |
| PowerShell 中文参数被拆开 | 给中文参数加双引号，例如 `--goal "提取电影名称、评分"` |
| PowerShell 多行命令报错 | 使用反引号 `` ` `` 续行，或优先写成单行 |
| URL 无法打开 | 检查是否写完整协议，例如 `https://example.com` |
| `--output task.json` 写到意外目录 | 先 `cd` 到工作目录，或使用绝对路径 |
| `npm install -g ...@latest` 卡住或被终止 | 使用固定版本：`npm install -g bazhuayu-cli@0.1.28` |
| `doctor` 某项失败 | 根据 `ok: false` 的检查项修复 Chrome、引擎文件或浏览器依赖 |
