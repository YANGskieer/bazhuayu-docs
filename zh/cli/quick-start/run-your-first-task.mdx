---
title: 运行第一个任务
description: 使用 run-url 直接采集新 URL，或运行已有八爪鱼任务并导出采集数据。
---

> 按你的场景选择路径：新 URL 用 `run-url --auto`，已有任务用 `run <taskId>`，需要精细控制时先 `recognize` 生成任务文件。

开始前请已完成 [安装](/zh/cli/quick-start/installation) 与 [登录](/zh/cli/quick-start/get-api-key-and-log-in)。

## 路径一：新 URL 快速采集

如果没有现成任务，直接从 URL 启动采集：

```bash
octopus run-url https://movie.douban.com/explore --auto --goal "提取电影名称和评分" --max-rows 20
```

常用选项：

```bash
octopus run-url URL --auto
octopus run-url URL --auto --goal "提取商品名称、价格、链接"
octopus run-url URL --auto --max-rows 20
octopus run-url URL --auto --json
octopus run-url URL --auto --jsonl
```

`run-url` 适合快速试采、探索新页面、或让 AI 自动识别列表、详情页与分页逻辑。

## 路径二：识别 → 校验 → 运行

如果需要先生成可复用任务文件：

```bash
octopus recognize https://movie.douban.com/explore --auto --goal "提取电影名称、评分、导演、年份" --output task.json
octopus task validate --task-file task.json
octopus run recognized_movie.douban.com --task-file task.json --max-rows 20
```

识别阶段可根据输出候选区域选择更精准的方案：

```bash
octopus recognize URL --select protected_smart_1 --output task.json
```

## 路径三：运行已有任务

先列出任务：

```bash
octopus task list
octopus task list --keyword 新闻 --page 1 --page-size 10
octopus task list --json
```

从结果中记下 **任务 ID**（`taskId`）。

查看与校验任务：

```bash
octopus task inspect <taskId>
octopus task validate <taskId>
```

本地运行：

```bash
octopus run <taskId>
octopus run <taskId> --headless
octopus run <taskId> --max-rows 100
octopus run <taskId> --detach
octopus run <taskId> --output ./runs
```

后台运行后可查看、暂停、恢复或停止：

```bash
octopus local status <taskId>
octopus local pause <taskId>
octopus local resume <taskId>
octopus local stop <taskId>
octopus local cleanup
```

云端运行：

```bash
octopus cloud start <taskId>
octopus cloud status <taskId>
octopus cloud history <taskId>
octopus cloud stop <taskId>
```

## 导出数据

查看历史：

```bash
octopus data history <taskId>
octopus data history <taskId> --local
octopus data history <taskId> --cloud
```

导出结果：

```bash
octopus data export <taskId> --format xlsx --file result.xlsx
octopus data export <taskId> --local --format csv --file result.csv
octopus data export <taskId> --lot-id <lotId> --format json
```

支持格式：`xlsx`、`csv`、`html`、`json`、`xml`。
