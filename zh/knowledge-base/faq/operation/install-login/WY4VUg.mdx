客户端数，也就是客户端同时在线数，每台电脑只支持打开一个八爪鱼客户端。同一个账号根据套餐不同可以同时登陆的不同电脑数量。详见：八爪鱼价格[https://www.bazhuayu.com/plan](https://www.bazhuayu.com/plan)
