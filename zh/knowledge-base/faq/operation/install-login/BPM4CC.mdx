软件安装后无法启动，可能有多种情况：

 

1、正常安装，但点击八爪鱼图标无法启动软件或启动后报错，如无法链接网络，服务错误等。此时请有优先检查是否开代理或VPN；如有开启，请关闭后再重新启动软件。

 

2、在进行八爪鱼采集器的安装时，如果首次安装成功，在启动时提示“Windows正在配置八爪鱼采集器，请稍后”，并且之后出现提示“安装时发生严重错误”。同时您的电脑上有360安全卫士等类似的电脑安全防护软件正在运行，很可能是因为这类安全防护软件错误的删除了八爪鱼运行所需要的文件，这种情况下，请退出杀毒软件，重新安装八爪鱼采集器即可。

 

3、请尝试卸载掉八爪鱼，然后删除C:\Users\\\{用户名\}\AppData\Local 路径下的OctopusSoftV8和OctopusSoft文件夹，然后从官网下载最新版本的客户端重新安装。\{用户名\}指Windows系统的用户名，默认是Administrator 如下图：![image.png](/assets/knowledge-base/faq/BPM4CC/01-6541ee4bbcc24.png)
