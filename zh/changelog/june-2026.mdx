# 2026 年 6 月发布说明

> **更新日期：** 2026 年 6 月 10 日  
> MCP 五大 AI 客户端完整中文对接教程上线；站点根路径与更新日志入口路由优化。

## 新功能

### MCP 服务 — AI 客户端完整对接教程

ChatGPT、Claude、Cursor、VS Code、Gemini 五个平台的 MCP 对接页由外部跳转改为 **站内完整中文分步指南**，每篇均含：

- 前置条件、MCP 服务地址（`https://mcp.bazhuayu.com`）与 OAuth 认证说明
- 分步操作截图（安装插件 / 添加 MCP 服务器 / 登录授权等）
- 配置示例（如 Cursor / VS Code 的 `settings.json`、Gemini CLI 的 OAuth `clientId` 等）
- 接入后可用的能力说明与常见问题

| 平台 | 文档 |
| --- | --- |
| ChatGPT | [MCP 对接 ChatGPT](/zh/mcp/integrations/chatgpt) |
| Claude | [MCP 对接 Claude](/zh/mcp/integrations/claude) |
| Cursor | [MCP 对接 Cursor](/zh/mcp/integrations/cursor) |
| VS Code | [MCP 对接 VS Code](/zh/mcp/integrations/vscode) |
| Gemini | [MCP 对接 Gemini](/zh/mcp/integrations/gemini) |

各教程统一说明：导出格式为 **EXCEL / CSV / JSON / 数据库**；云采集可能消耗八爪鱼账户余额，具体取决于所用模板。
